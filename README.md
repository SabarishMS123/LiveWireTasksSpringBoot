# LiveWireTasksSpringBoot

This repository contains a collection of Java and Spring Boot practice tasks, covering core concepts like dependency injection, autowiring, servlets, JSP, and mini-project implementations.

---

## 📂 Project Contents

### 🔹 Spring Core (Autowiring)
- **AutoWiring.zip**  
  Demonstrates basic autowiring in Spring.

- **AutowireByName1.zip**  
  Example of autowiring using *byName* strategy.

- **AutowireByconstructor.zip**  
  Shows dependency injection using constructor-based autowiring.

- **Autowiring2.zip**  
  Advanced autowiring examples and configurations.

---

### 🔹 Spring Boot Projects
- **FileManagerSpringBootTaskPart2.zip**

- **LivewireTasksformxl.zip**  
  Contains LiveWire-related Spring Boot task implementations.

- **SPRINGTASKS.zip**  
  Collection of various Spring Boot exercises and mini-tasks.

### 🔹 JSP & Servlet Tasks
